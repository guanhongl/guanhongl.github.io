# Forget Formality
The first thing that stood out to me about Javascript was its simplicity. Javascript seems to drop the technical mumbo-jumbo I often find myself googling for other languages such as Java and C. Forget about the main function, header files, and type declarations; drop the formality when it comes to Javascript. Programming suddenly feels more lax and fun. Having prior knowledge in Java definitely helped as Javascript uses much of the same syntax. I think Javascript is a great language from a software engineering perspective. Not having to worry too much about syntax will allow engineers to focus on their program. Also, being a popular language and current helps I have only scratched the surface and there is so much more to learn, but I'm eager to continue my journey through Javascript.
